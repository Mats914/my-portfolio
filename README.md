# My Portfolio

Welcome to my personal portfolio! This project showcases my skills and projects using modern web technologies.

## Built With

- **Vite** – A fast frontend build tool
- **TypeScript** – Strongly typed JavaScript
- **React** – JavaScript library for building user interfaces
- **shadcn-ui** – A modern React component library
- **Tailwind CSS** – Utility-first CSS framework

This portfolio demonstrates my work, projects, and experience in web development. It is designed to be fully responsive and easy to navigate.

## Features

- Clean and modern user interface
- Responsive design for all devices
- Reusable React components
- Built with TypeScript for type safety
- Styled using Tailwind CSS

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Mats914/my-portfolio.git


2. Navigate to the project folder: cd my-portfolio
3. Install dependencies: npm install
4. Start the development server: npm run dev
